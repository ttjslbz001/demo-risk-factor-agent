## 输出契约
- 日志规范与错误码清单（文档）
- 关键错误分类：输入校验、规则加载、鉴权、网络超时、解析失败

## 实现约束
- 日志脱敏；不落地原始 JSON 与全文规则
- 分级日志：INFO（开始/完成）、WARN（可恢复）、ERROR（阻断）
